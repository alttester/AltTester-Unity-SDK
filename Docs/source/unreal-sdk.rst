AltTester® Unreal SDK
=====================

AltTester® Unreal SDK is a UI driven test automation tool that helps you
find objects in your application and interacts with them using tests written in C#,
Java, Python or Robot Framework. You can run your tests on real devices (mobile, PCs, etc.) or
inside the Unreal Editor.
